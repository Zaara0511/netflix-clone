# Netflix Clone 🎬

This project is a beginner-friendly Netflix homepage clone built using HTML and CSS. The main goal was to practice layout structuring, styling, and responsive design by recreating the look and feel of Netflix’s landing page.

## 🚀 Features

- Netflix-style homepage layout  
- Hero section with background image  
- Navigation bar design  
- Styled buttons and content sections  
- Responsive layout using CSS  
- Clean and organized code structure  

## 🛠️ Technologies Used

- HTML5  
- CSS3
